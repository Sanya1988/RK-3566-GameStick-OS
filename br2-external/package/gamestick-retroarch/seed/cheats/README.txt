Эта папка предназначена для читов RetroArch.

Что сюда класть:
- файлы читов в формате .cht
- подпапки с читами, если вы хотите хранить их по системам, ядрам или отдельным играм

Рекомендуется:
- использовать распакованные .cht-файлы
- сохранять структуру папок из официального архива
- не переименовывать файлы без необходимости

Официальный архив читов RetroArch:
https://buildbot.libretro.com/assets/frontend/cheats.zip

Примечание:
- в этой сборке онлайн-обновление RetroArch отключено
- поэтому читы нужно скачивать и копировать сюда вручную
