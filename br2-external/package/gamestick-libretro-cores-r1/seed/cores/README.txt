ВНИМАНИЕ

Эта папка содержит ядра RetroArch, необходимые для запуска игр.

Не рекомендуется:
- удалять файлы из этой папки
- заменять ядра другими версиями
- переименовывать файлы
- вручную копировать сюда случайные ядра из интернета

Изменение или удаление содержимого этой папки может привести к неработоспособности системы
или к тому, что отдельные игры перестанут запускаться.
