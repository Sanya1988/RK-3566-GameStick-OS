ES-DE ищет медиа не по тегам внутри gamelist.xml, а по имени ROM-файла.

Рабочий формат:
- downloaded_media/<system>/covers/<rom_basename>.png
- downloaded_media/<system>/screenshots/<rom_basename>.png
- downloaded_media/<system>/fanart/<rom_basename>.png
- downloaded_media/<system>/marquees/<rom_basename>.png

Для demo-игр в сборке используются официальные страницы:
- Miniplanets: https://sik.itch.io/miniplanets
- Sliding Blaster: https://novasquirrel.itch.io/sliding-blaster
- Super Rambler: https://torte00.itch.io/super-rambler
