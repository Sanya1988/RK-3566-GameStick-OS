Эта папка предназначена для тем ES-DE.

Как добавлять темы:
- каждая тема должна лежать в отдельной подпапке
- внутри папки темы должен быть файл theme.xml
- копируйте тему целиком, вместе со всеми файлами и подпапками

Официальный список тем ES-DE:
https://gitlab.com/es-de/themes/themes-list

Примечание:
- после добавления новой темы выберите её в настройках оформления ES-DE
