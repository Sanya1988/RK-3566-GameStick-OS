Эта папка используется экранной заставкой ES-DE.

Сюда можно добавлять свои изображения.

Требования:
- формат файлов: JPG или PNG
- рекомендуемое разрешение: 1920x1080

Сколько изображений лежит в этой папке, столько и будет использовать заставка.
Смена изображений происходит автоматически в режиме слайд-шоу.
